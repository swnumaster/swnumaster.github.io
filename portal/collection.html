EasyTrans Server
(Centnet 2004-2006)

EasyTrans, short for Easily Transferring, was a relay server for large concurrent voice data stream during VOIP communication, making gateways located in LAN communicate smoothly with VOIP servers across all kinds of firewalls. Due to the limited volume of traditional approach through the Linux application layer, a single server could not support enough channels. By contrast, the EasyTrans server handled data in the kernel layer of Linux, which made it much more efficient, so the capacity of a single server increased by 400%.

My Major Achievements

I designed and developed EasyTrans Server, lifting server capacity from 200 channels to 1000 in a single server. Finally, its realization method obtained a China National Patent.

Tech Stack

C/C++, Linux, Linux kernel, gcc, g++, gdb, MySQL, memcache etc.
